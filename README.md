# Auto with valid number Parser

Этот препозиторий реализует скрипт парсинга интернет магаизина автомобилей с валидным отображением номеров.

number
![01_number.jpg](assets%2F01_number.jpg)

## How to run?

- Python 3.9

```bash
python main.py
```

После запуска команды, создатся папка `downloaded_images` содержащая все обнаруженные изображения на сайте.
